# 박명원 자기소개 사이트

간호학과 2학년 박명원의 개인 소개 페이지입니다.

🔗 https://auddnjs4444.github.io

## 구성

| 파일 | 설명 |
| --- | --- |
| `index.html` | 페이지 구조 (소개 · 취미 · 목표 · 연락) |
| `style.css` | 스타일 (색상·타이포·반응형·애니메이션) |
| `script.js` | 스크롤 등장 효과, 네비게이션, 연도 자동 갱신 |

## 수정하는 법

내용은 `index.html`만 고치면 됩니다. 색상을 바꾸고 싶다면 `style.css` 맨 위의
`:root` 안 값을 수정하세요.

```css
--sage: #4f8a72;  /* 포인트 색 (초록) */
--clay: #c96a52;  /* 포인트 색 (주황) */
```

저장 후 커밋·푸시하면 1~2분 안에 사이트에 반영됩니다.

```bash
git add . && git commit -m "내용 수정" && git push
```
